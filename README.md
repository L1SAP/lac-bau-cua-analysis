# Lunar New Year Simulation

A Python simulation of the traditional Vietnamese Lunar New Year game **Lắc Bầu Cua**.

A dice betting game where players bet on six symbols (Deer, Gourd, Rooster, Fish, Crab, Shrimp) and win based on how many dice land on their chosen symbol.

This project lets you build a custom betting strategy, simulate multiple sessions, and export results to CSV for analysis.

## Features
- Custom betting strategy input
- Simulate multiple rounds and sessions
- Export results to CSV for analysis
