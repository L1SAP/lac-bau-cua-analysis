# Lắc Bầu Cua analysis

A simulation of the traditional Vietnamese Lunar New Year game **Lắc Bầu Cua**. Using python, pandas, and jupyter notebook for analysis.

Players bet on six symbols (Deer, Gourd, Rooster, Fish, Crab, Shrimp) and win based on how many dice matched their pick.

The house edge makes it impossible to win in the long term, but in a typical 10 round session (which is about the # of rounds my family usually plays) **can the right betting strategy survive all 10 rounds and walk away with profit after a session?**

This project simulates 1,000 sessions across different strategies, analyzes survival rates, profit distributions, and expected value.
